# Build app

- [Install python v.3+ and pip](https://www.python.org/downloads/)
- Install pyinstaller and *ECDSA* lib:
    ```shell
    pip install pyinstaller ecdsa
    ```
- Build one-file executable Windows/MacOS/Linux:
    ```shell
    python build.py
    ```
- Executable would be in *dist* folder
